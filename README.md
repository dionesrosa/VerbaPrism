Verba Prism é um userscript para Tampermonkey que utiliza a API da Groq para revisar, corrigir e aprimorar textos selecionados em qualquer página da web.

O script permite configurar uma chave de API Groq, escolher o modelo de IA desejado e enviar trechos de texto selecionados para processamento. Após a análise, o texto aprimorado é exibido em uma janela interativa, permitindo copiar o resultado ou substituir o texto original diretamente na página.

Principais funcionalidades:

• Correção ortográfica, gramatical e de pontuação.
• Aprimoramento de clareza, fluidez e legibilidade.
• Preservação da estrutura e estilo original do texto.
• Integração com múltiplos modelos da Groq.
• Interface visual dark, moderna e elegante, com nuances de rosa forte para elementos principais.
• Funcionamento em qualquer site acessado pelo navegador.
• Armazenamento local das configurações através do Tampermonkey.

O objetivo do Verba Prism é atuar como um assistente de escrita universal, permitindo que o usuário melhore rapidamente qualquer texto sem precisar copiar e colar conteúdo entre diferentes ferramentas.
